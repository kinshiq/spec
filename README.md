# Kinshiq Specifications

Kinshiq is a machine-first system for representing, analyzing, and reasoning about relationships between individuals.

This repository contains the normative Kinshiq specifications.

## Kinshiq Core

The Kinshiq Core Specification defines the MRCA-based encoding of biological pedigree relationships using symbolic expressions called **qodes**.

Editor’s draft:

→ [Kinshiq Core Specification](kinshiq-core.md)

## Text Formats

The repository also contains draft shared text-format specifications for genealogy-oriented tree representations.
